Welcome to MMFT OoC Designer's documentation!
=============================================

MMFT OoC Designer is a tool for designing Organ-on-Chip (OoC) devices developed as part of the `Munich MicroFluidic Toolkit <https://www.cda.cit.tum.de/research/microfluidics/munich-microfluidics-toolkit/>`_ (*MMFT*) by the `Chair for Design Automation <https://www.cda.cit.tum.de/>`_ at the `Technical University of Munich <https://www.tum.de>`_.

We recommend you to start with the :doc:`Step-by-Step Tutorial <../Tutorial>`.
